BFS и кратчайшие пути в невзвешенном графе

Данный проект реализует обход графа в ширину (BFS) и поиск кратчайших путей в невзвешенном графе.
Программа поддерживает:

Загрузку графа из файла в виде матрицы смежности или матрицы инцидентности (конвертируется автоматически в смежность).

Генерацию случайного графа с произвольным числом вершин и рёбер.

Пошаговую анимацию обхода графа с подсветкой вершин и рёбер.

Вывод таблицы с родителями, кратчайшими расстояниями и порядком обхода.

Установка и зависимости

Для корректной работы проекта требуется Python 3.8+ и следующие библиотеки:

numpy

networkx

matplotlib
